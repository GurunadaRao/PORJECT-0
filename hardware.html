Hardware Components

Core components of the Smart Gas Detection and Fire Prevention System

ESP32 Microcontroller

Main control unit for processing sensor data and managing system responses.

  • Dual-core Xtensa LX6 microprocessor
  • 240 MHz clock frequency
  • 512 KB RAM
  • 4MB Flash Memory
  • WiFi 802.11 b/g/n
  • Bluetooth 4.2
  • 34 GPIO pins
  • 12-bit ADC
  • Operating Voltage: 3.3V

MQ6 Gas Sensor

High-sensitivity sensor for detecting LPG, butane, and propane gas leaks.

  • Detection Range: 200-10000 ppm
  • Operating Voltage: 5V
  • Heater Consumption: <800mW
  • Response Time: <10s
  • Operating Temperature: -10°C to 50°C
  • Analog Output
  • High Sensitivity to LPG, Butane, Propane
  • Long Life and Stable Performance

Servo Motor

Controls gas valve for emergency shutoff during leak detection.

  • Operating Voltage: 4.8V - 6V
  • Stall Torque: 1.8 kg/cm
  • Speed: 0.1s/60°
  • Rotation: 180°
  • PWM Control
  • Weight: 9g
  • Operating Temperature: 0°C to 55°C

Exhaust Fan

Automatic ventilation system for gas dispersion during leaks.

  • Voltage: 220V AC
  • Power: 25W
  • Air Flow: 100 CFM
  • Noise Level: <40 dB
  • Size: 6 inches
  • Speed: 1200 RPM
  • Material: ABS Plastic

Relay Module

Controls high-voltage components like exhaust fans and alarms.

  • Operating Voltage: 5V DC
  • Maximum Load: 10A 250V AC
  • Number of Channels: 4
  • Switching Time: <10ms
  • Optical Isolation
  • LED Indicators
  • Low-level Trigger
